## Basics

Tray will appear whenever included in DOM, and you can use `onCloseComplete` to clean up when the tray is dismissed.

It is recommended to pin the tray to the right side of the screen on web.

```jsx live
function BasicTray() {
  const [visible, setVisible] = useState(false);
  const handleOpen = () => setVisible(true);
  const handleClose = () => setVisible(false);

  return (
    <VStack gap={2}>
      <Button onClick={handleOpen}>Open Tray</Button>
      {visible && (
        <Tray
          title="Example Title"
          pin="right"
          onCloseComplete={handleClose}
          footer={({ handleClose }) => (
            <PageFooter
              borderedTop
              justifyContent="flex-end"
              action={<Button onClick={handleClose}>Close</Button>}
            />
          )}
        >
          <VStack gap={2} paddingBottom={2}>
            <Text>Basic content.</Text>
          </VStack>
        </Tray>
      )}
    </VStack>
  );
}
```

## Responsive

Use `useBreakpoints` to create a Tray that adapts between mobile and desktop layouts. On mobile, the tray pins to the bottom with a full-width button. On desktop, it pins to the right with a standard button.

```jsx live
function ResponsiveTray() {
  const [visible, setVisible] = useState(false);
  const { isPhone } = useBreakpoints();
  const handleOpen = () => setVisible(true);
  const handleClose = () => setVisible(false);

  return (
    <VStack gap={2}>
      <Button onClick={handleOpen}>Open Responsive Tray</Button>
      {visible && (
        <Tray
          title="Responsive Tray"
          pin={isPhone ? 'bottom' : 'right'}
          onCloseComplete={handleClose}
          footer={({ handleClose }) => (
            <PageFooter
              borderedTop
              action={
                <Button block={isPhone} onClick={handleClose}>
                  Close
                </Button>
              }
            />
          )}
        >
          <VStack gap={2}>
            <Text>
              This tray adapts to your screen size. On mobile, it pins to the bottom with a
              full-width close button. On desktop, it pins to the right.
            </Text>
            {Array.from({ length: 15 }, (_, i) => (
              <Box key={i} background="bgSecondary" borderRadius={200} padding={2}>
                <Text>Item {i + 1}</Text>
              </Box>
            ))}
          </VStack>
        </Tray>
      )}
    </VStack>
  );
}
```

#### Prevent Dismiss

Using `preventDismiss` allows you to disable the default close functionality. Using this will prevent users from seeing the default close button and they won't be able to click outside of the Tray to close.

Children of Tray can still use `handleClose`.

```jsx live
function BasicTray() {
  const [visible, setVisible] = useState(false);
  const handleOpen = () => setVisible(true);
  const handleClose = () => setVisible(false);

  return (
    <VStack gap={2}>
      <Button onClick={handleOpen}>Open Tray</Button>
      {visible && (
        <Tray title="Example Title" pin="right" preventDismiss onCloseComplete={handleClose}>
          {({ handleClose }) => (
            <VStack gap={2}>
              <Text>You can only close this tray by clicking the close button.</Text>
              <Button onClick={handleClose}>Close</Button>
            </VStack>
          )}
        </Tray>
      )}
    </VStack>
  );
}
```

```jsx live
    onClear,

    return (
```

### Floating Tray

A composed `FloatingTray` component that wraps `Tray` with customizable floating behavior.

```jsx live
function FloatingTrayExample() {
  /**
   * FloatingTray - A composed Tray component with floating panel styling
   *
   */
  function FloatingTray({ pin = 'right', offset = '2', borderRadius = '600', styles, ...props }) {
    const theme = useTheme();

    const offsetPx = theme.space[offset];
    const borderRadiusVar = `var(--borderRadius-${borderRadius})`;

    const floatingInsets = useMemo(() => {
      switch (pin) {
        case 'right':
          return { top: offsetPx, bottom: offsetPx, right: offsetPx };
        case 'left':
          return { top: offsetPx, bottom: offsetPx, left: offsetPx };
        case 'top':
          return { top: offsetPx, left: offsetPx, right: offsetPx };
        case 'bottom':
          return { bottom: offsetPx, left: offsetPx, right: offsetPx };
        default:
          return { top: offsetPx, bottom: offsetPx, right: offsetPx };
      }
    }, [pin, offsetPx]);

    return (
      <Tray
        {...props}
        pin={pin}
        styles={{
          ...styles,
          container: {
            ...floatingInsets,
            // All corners rounded since the tray is floating (not flush to edge)
            borderRadius: borderRadiusVar,
            ...styles?.container,
          },
        }}
      />
    );
  }

  function Example({ label = 'Bottom (default)', direction = 'right' }) {
    const [visible, setVisible] = useState(false);
    const handleOpen = () => setVisible(true);
    const handleClose = () => setVisible(false);

    return (
      <VStack gap={2}>
        <Button onClick={handleOpen}>Open {direction}</Button>
        {visible && (
          <FloatingTray
            title="Floating Panel"
            pin={direction}
            offset="2"
            borderRadius="600"
            onCloseComplete={handleClose}
            footer={({ handleClose }) => (
              <PageFooter
                borderedTop
                justifyContent="flex-end"
                action={<Button onClick={handleClose}>Close</Button>}
              />
            )}
          >
            <VStack gap={2} paddingBottom={2}>
              <Text>This Tray is floating and not touching the edge of the screen.</Text>
            </VStack>
          </FloatingTray>
        )}
      </VStack>
    );
  }

  return (
    <HStack gap={2} flexWrap="wrap">
      <Example />
      <Example direction="left" label="Left" />
      <Example direction="top" label="Top" />
      <Example direction="bottom" label="Bottom" />
    </HStack>
  );
}
```

### Using Ref to Control the Tray

:::tip Accessibility tip

**Trigger Focus**

A `ref` to the trigger that opens the tray, along with an `onClosedComplete` method to reset focus on the trigger when the tray closes, needs to be wired up for accessibility (see code example below).

:::

You can also control the Tray using a ref, which provides a `close()` method:

```tsx live
function TrayWithRef() {
  const [visible, setVisible] = useState(false);
  const trayRef = useRef<DrawerRefBaseProps>(null);
  const triggerRef = useRef<HTMLButtonElement>(null);

  const handleOpen = () => setVisible(true);
  const handleClose = () => {
    setVisible(false);
    triggerRef.current?.focus();
  };

  return (
    <VStack gap={2}>
      <Button ref={triggerRef} onClick={handleOpen}>
        Open Tray
      </Button>
      {visible && (
        <Tray ref={trayRef} title="Ref Controlled Tray" onCloseComplete={handleClose}>
          <VStack gap={2}>
            <Text>Control this tray using the ref.</Text>
            <Button onClick={() => trayRef.current?.close()}>Close</Button>
          </VStack>
        </Tray>
      )}
    </VStack>
  );
}
```

### Multiple Overlay Flow

When transitioning between overlays, ensure proper dismounting using `onCloseComplete`:

```tsx live
function TrayToModalFlow() {
  const [isTrayVisible, setIsTrayVisible] = useState(false);
  const [isModalVisible, setIsModalVisible] = useState(false);

  const openTray = () => setIsTrayVisible(true);
  const closeTray = () => setIsTrayVisible(false);
  const openModal = () => setIsModalVisible(true);
  const closeModal = () => setIsModalVisible(false);

  const handleTrayClose = useCallback(() => {
    closeTray();
    openModal();
  }, []);

  return (
    <VStack gap={2}>
      <Button onClick={openTray}>Start Flow</Button>
      {isTrayVisible && (
        <Tray title="First Step" onCloseComplete={handleTrayClose}>
          {({ handleClose }) => (
            <VStack gap={2}>
              <Text>Click below to continue to the modal</Text>
              <Button onClick={handleClose}>Continue to Modal</Button>
            </VStack>
          )}
        </Tray>
      )}
      <Modal visible={isModalVisible} onRequestClose={closeModal}>
        <ModalHeader title="Second Step" />
        <ModalBody>
          <VStack gap={2}>
            <Text>This is the second step in the flow.</Text>
            <Button onClick={closeModal}>Finish</Button>
          </VStack>
        </ModalBody>
      </Modal>
    </VStack>
  );
}
```

Note: The Tray component is an elevated container that is pinned to the bottom of the screen and provides a standardized way to present bottom sheets in your web application. Key points:

- Use `onCloseComplete` for cleanup when the tray is dismissed
- Children can be either a React node or a render function that receives a `handleClose` function
- The `ref` provides `open()` and `close()` methods for controlling the tray
- When transitioning between overlays, ensure proper dismounting using lifecycle methods

```tsx live
        <Button onClick={() => setPin('bottom')}>Bottom (default)</Button>
        <Button onClick={() => setPin('top')}>Top</Button>
        <Button onClick={() => setPin('left')}>Left</Button>
        <Button onClick={() => setPin('right')}>Right</Button>
      </HStack>
      {pin && (
        <Tray title={`Pinned ${pin}`} pin={pin} onCloseComplete={handleClose}>
          <VStack gap={2}>
            <Text>This tray is pinned to the {pin} of the screen.</Text>
            <Text>
              Side-pinned trays (left/right) are useful for navigation drawers or settings panels.
            </Text>
          </VStack>
        </Tray>
      )}
    </VStack>
  );
}
```

### Custom Styling

You can customize the appearance of different parts of the Tray using the `styles` and `classNames` props.

The following keys are available:

- `root` - The root container element
- `overlay` - The overlay backdrop
- `container` - The animated sliding container
- `header` - The header section
- `title` - The title text
- `content` - The content area
- `footer` - The footer section

```tsx live
function StyledTray() {
  const [visible, setVisible] = useState(false);
  const handleOpen = () => setVisible(true);
  const handleClose = () => setVisible(false);

  return (
    <VStack gap={2}>
      <Button onClick={handleOpen}>Open Styled Tray</Button>
      {visible && (
        <Tray
          title="Custom Styled Tray"
          onCloseComplete={handleClose}
          styles={{
            container: { borderRadius: 24 },
            header: { background: 'var(--color-bgSecondary)' },
            content: { padding: 'var(--space-2)' },
          }}
        >
          <VStack gap={2}>
            <Text>This tray has custom styles applied to its container, header, and content.</Text>
            <Text>
              You can also use the <code>classNames</code> prop to apply CSS classes for more
              advanced styling with Linaria or other CSS-in-JS solutions.
            </Text>
          </VStack>
        </Tray>
      )}
    </VStack>
  );
}
```
